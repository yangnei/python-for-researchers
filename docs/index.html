Python for Researchers

A fast, hands-on path from "never coded" to wrangling your own research data — adapted from Harvard's CS50P and front-loaded with the language quirks that trip beginners up.

Every session runs real Python in your browser — edit the code and press Run. Nothing to install.

9 one-hour sessions Interactive code Cheat sheets Self-checking quizzes
💡 How the live code works: the first time you press Run, your browser downloads a small Python engine (Pyodide, ~10 MB) and then runs everything locally — your code never leaves your machine. Your progress (✓ marks) is saved in this browser only.
